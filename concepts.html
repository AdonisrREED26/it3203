Essential Domain Concepts

Review these critical core concepts and technical terms underlying modern web systems, infrastructure models, and application protocols:

1. Latency
The total time delay or duration it takes for data packet payloads to travel across a network from their origin point to their destination.
2. DOM (Document Object Model)
An API layout standard that handles structured HTML/XML documents as nodes in a tree system, allowing dynamic script languages to alter structural content.
3. CDN (Content Delivery Network)
A geographically distributed infrastructure network of proxy caching servers aimed at providing content rapidly by reducing geographic gaps.
4. Hydration
A client-side layout process where user interface scripts scan pre-rendered static markup and attach active event handlers directly to them.
5. State Management
The architectural standard of recording, modifying, and tracking visual states and dataset updates consistently across a user application layout.
6. Multiplexing
The modern network capacity to transmit multiple independent data request and response streams simultaneously over a single shared connection link.
